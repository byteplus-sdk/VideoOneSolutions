# BytePlus RTC SDK

## Product Introduction

BytePlus RTC enables highly reliable, concurrent and low latency real-time communication across
the globe and provides a service that allows various forms of online interaction.
By integrating BytePlus RTC SDK into your application and calling the API, you can quickly
start audio & video chats, online classes, live streaming and virtual meetings, and use them 
in various scenes including cloud gaming and cloud rendering.[Learn More](https://docs.byteplus.com/en/docs/byteplus-rtc/docs-66812)...

>BytePlus RTC SDK supports all major platforms including Web, Android, iOS, Windows, Linux, macOS, as well as Flutter, Mini Programs, Electron, and more，
> [Learn more](https://docs.byteplus.com/en/docs/byteplus-rtc/docs-75707)...
